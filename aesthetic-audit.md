# Home 页面美学审计

## 桌面端问题
1. **导航栏缺乏精致感** — 纯文字导航没有 hover 下划线或指示器，"免费检测"按钮与整体风格稍显突兀
2. **左侧内容垂直节奏不够** — 标题、描述、按钮、链接、统计数据之间的间距不够有层次感，信息密度偏高
3. **"咨询专家"按钮是占位功能** — 但视觉权重和"开始检测"一样大，容易误导
4. **"护肤日历""历史记录"链接** — 字号太小，视觉上像被遗忘的元素
5. **浮动卡片可以更精致** — 综合评分卡片和维度分析卡片的毛玻璃效果可以更柔和
6. **footer 太简陋** — "服务运行中"的脉冲点缺乏品质感

## 移动端问题
1. **上传卡片太空旷** — 虚线边框 + 大面积空白，看起来像未完成的占位符
2. **快捷提问胶囊颜色差异太微妙** — 在实际屏幕上四种暖色几乎看不出区别
3. **中间大段空白** — 快捷提问和底部聊天按钮之间有巨大空白（mt-auto），页面不够充实
4. **缺少视觉焦点** — 没有任何图片或插图，整个页面纯文字+图标，缺乏情感连接
5. **"或者直接问我"标题太小** — 几乎看不见

## 优化方案
### 移动端
- 上传卡片改为实心渐变背景 + 更大的图标 + 更有质感的设计
- 快捷提问胶囊加上小 emoji 图标，颜色对比度加大
- 去掉 mt-auto，在快捷提问下方加一个轻量的"最近检测"或"芯颜小贴士"卡片填充空间
- 底部聊天按钮保持不变

### 桌面端  
- 导航加 hover 下划线动画
- 减弱"咨询专家"按钮视觉权重
- 统计数据加分隔线
- 浮动卡片加更柔和的阴影
